# Issue 0006 - Q006 測試配置錯誤

## 題目資訊
- **題目 ID**: DIS-M006 (Q006)
- **標題**: Virtual Display 釋放時的空指標處理缺陷
- **失敗步驟**: C4 (CTS 測試執行)

## 問題類型
測試配置錯誤

## 問題描述
meta.json 中的 CTS 測試類名錯誤：
- **錯誤的**: `android.hardware.display.cts.VirtualDisplayTest#testPrivateVirtualDisplay`
- **正確的**: `android.display.cts.VirtualDisplayTest#testPrivateVirtualDisplay`

此外，CTS 14_r7 版本可能不包含該測試，執行結果為 0 Tests。

## 錯誤訊息
```
java.lang.ClassNotFoundException: android.hardware.display.cts.VirtualDisplayTest
Total Tests       : 0
PASSED            : 0
FAILED            : 0
```

## 相關檔案
- `domains/display/medium/Q006/meta.json`
- `domains/display/medium/Q006/bug.patch`

## 建議修復
1. 修正 meta.json 中的 `cts_test` 為 `android.display.cts.VirtualDisplayTest#testPrivateVirtualDisplay`
2. 確認 CTS 14_r7 是否包含此測試
3. 如需要，更新 CTS 版本或選擇其他能偵測此 bug 的測試
